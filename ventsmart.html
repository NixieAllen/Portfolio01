SMART Ventilation System

This project uses an ESP32 to monitor air quality and motion using sensors like MQ2, MQ135, and a dust sensor. The system activates a fan at different speeds based on readings and motion detection. Data is sent to Firebase and optionally integrated with ThingSpeak for cloud analytics.

Key Features:

  • Real-time gas & dust monitoring
  • Motion-triggered fan control
  • Firebase cloud data sync
  • Alert system for critical air levels

System Overview

The system collects data from gas and dust sensors and uses it to intelligently control a fan. If poor air quality is detected, the fan turns on at high speed. If only motion is detected, it runs on low speed. Alerts are pushed to Firebase, and all sensor data is regularly synced.
