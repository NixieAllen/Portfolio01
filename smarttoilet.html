SMART Toilet System

Description: This Arduino-based project uses an ESP32 microcontroller to collect data from various environmental sensors and upload it to Firebase Realtime Database. It can be used in smart monitoring systems for homes, schools, or coastal areas.

Features

  • Temperature and Humidity Monitoring (DHT22)
  • Motion Detection (PIR Sensor)
  • Gas/Smoke Detection (MQ135)
  • Distance Measurement (Ultrasonic Sensor)
  • Color Detection (TCS3200)
  • Servo Control Based on Sensor Inputs
  • Data Upload to Firebase

System Overview

The system designed for an ESP32-based environmental monitoring system. It reads real-time data from sensors (such as the DHT22 for temperature and humidity) and uploads the values to a Firebase Realtime Database over Wi-Fi. This setup allows for remote monitoring of environmental conditions, making it suitable for smart homes, weather stations, or coastal observation systems. It can also serve as a foundation for more advanced IoT (Internet of Things) applications that require cloud-based data storage and access.
